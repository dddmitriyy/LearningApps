# Notes

Приложение, которое позволяет пользователям сохранять заметки и добавлять тэги к ним.

Демонстрирует работу с библиотекой <code>Room</code>, оберткой для базы данных <code>SQLite</code>

Также в приложении показана работа с менеджером фрагментов, материальным компонентом <code>Chip</code> и сервисом
<code>ClipboardManager</code> для копирования в буфер обмена текста.
